# Diabetes Risk Predictor

An ML-powered web app that predicts diabetes risk from health parameters using a Random Forest classifier trained on the Pima Indians Diabetes Dataset.

**Tech Stack:** React + Vite · FastAPI · scikit-learn · Pima Indians Dataset  
**Deploy:** Frontend on Vercel · Backend on Render

Live: https://diabetes-risk-predictor-cyan.vercel.app/
---

## Local Setup

### Step 1 — Get the full dataset (recommended)
1. Go to https://www.kaggle.com/datasets/uciml/pima-indians-diabetes-database
2. Download `diabetes.csv`
3. Place it in the `ml/` folder

### Step 2 — Train the model
```bash
cd ml
pip install scikit-learn pandas numpy joblib
python train.py
```
You should see ~78% accuracy with the full dataset.
This generates 3 files in `backend/model/`.

### Step 3 — Run the backend
```bash
cd backend
pip install -r requirements.txt
uvicorn main:app --reload
# Runs on http://localhost:8000
# API docs at http://localhost:8000/docs
```

### Step 4 — Run the frontend
```bash
cd frontend
npm install
cp .env.example .env
npm run dev
# Runs on http://localhost:5173
```

## Deployment

### Backend on Render
1. Push repo to GitHub
2. Go to render.com → New Web Service
3. Connect repo, set Root Directory to `backend`
4. Build Command: `pip install -r requirements.txt`
5. Start Command: `uvicorn main:app --host 0.0.0.0 --port $PORT`
6. **Important:** You must commit the `model/` folder to GitHub (it contains the trained model)

### Frontend on Vercel
1. Go to vercel.com → Import repo
2. Set Root Directory to `frontend`
3. Add env variable: `VITE_API_URL=https://your-render-url.onrender.com`
4. Deploy

## API Endpoints

| Method | Endpoint | Description |
|--------|----------|-------------|
| GET | / | API status |
| POST | /predict | Run prediction |
| GET | /health | Health check |
| GET | /docs | Swagger UI (auto-generated) |

### Sample request
```json
POST /predict
{
  "pregnancies": 2,
  "glucose": 120,
  "blood_pressure": 72,
  "skin_thickness": 20,
  "insulin": 80,
  "bmi": 28.5,
  "diabetes_pedigree": 0.45,
  "age": 30
}
```

### Sample response
```json
{
  "risk_percentage": 34.5,
  "risk_level": "Moderate",
  "prediction": 0,
  "top_factors": [
    {"name": "Glucose", "importance": 28.3, "value": 120.0},
    {"name": "BMI", "importance": 17.2, "value": 28.5}
  ],
  "advice": "Some risk factors detected. Consider consulting a doctor."
}
```
